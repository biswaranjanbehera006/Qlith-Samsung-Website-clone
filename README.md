# 🌐 Samsung India Website Clone

A sleek, modern, and fully responsive **Samsung India Website Clone** built using **HTML**, **CSS**, **Tailwind CSS**, and **JavaScript**. This project replicates the professional design of Samsung India’s website with interactive navigation, smooth animations, and fully responsive layouts — ideal for improving front-end development skills!

---

## 🚀 Live Demo

🔗 **Live Website:** [https://samsung-clone-qlith.netlify.app/](https://samsung-clone-qlith.netlify.app/)

---

## 🛠️ Tech Stack

- 🟠 **HTML5** — Markup Language
- 🔵 **CSS3** — Styling and Layout
- 🌬️ **Tailwind CSS** — Utility-first CSS Framework
- 🟡 **JavaScript** — Interactive Features
- 🌐 **Netlify** — Deployment Platform

---

## 🎁 Features

✅ **Responsive Design** — Mobile, Tablet, and Desktop compatible  
✅ **Modern Samsung Layout** — Inspired by the official Samsung India website  
✅ **Smooth Animations** — Eye-catching scroll and hover effects  
✅ **Interactive Navigation** — Dropdown menus and active link highlights  
✅ **Product Sections** — Mobiles, TVs, Refrigerators, and more  
✅ **Attractive Footer** — Clean and professional footer layout

---

## 📌 Sections Included

- ✅ **Hero Section** — Clean product showcase banners
- ✅ **Product Categories** — Organized categories (Mobile, TV, Refrigerator, etc.)
- ✅ **Offers & Deals** — Stylish promotional offers
- ✅ **Interactive Navigation Bar**
- ✅ **Contact & Footer Section**

```bash
git clone https://github.com/biswaranjanbehera006/Qlith-Samsung-Website-clone.git

````

### 2. Open Project Locally

Simply open the `index.html` file in your browser.

Use **Live Server Extension** in VS Code for a smoother development experience.
